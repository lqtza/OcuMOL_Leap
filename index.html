    3-D Visualization and Manipulation of Molecules with PyMOL, Leap Motion, and Oculus Rift

    This is an alpha release of Python software for PyMOL which can be used to visualize (Oculus Rift) and manipulate (Leap Motion) molecules in 3-D. This software has lots of dependencies and peculiarities; it may not work instantly for you (it may not ever work for you)! However, it has been tested on a lonely MacBook Pro and we know it works on that so that being said, feel free to download and play with it, if you can.

    Dependencies

    Head on over to the actual GitHub (https://github.com/lqtza/OcuMOL_Leap) for installation instructions.
